Main Description

UI Graphic Designer / UI Developer

 

Inputs:

1.    Review requirements files sent by client.

2.    Create prototype design

3.    Perform features will be used on ui development

4.    Graphic design package

Outputs:

1.    Ui branding

2.    Ui wireframe design

3.    Ui integration design

4.    Perform ui code review and smoke test

5.    Production test

Steps:

1-   Create prototypes for propose design requirement.

2-   Create 2 samples designs to get approved from client.

3-   Waiting for client response and PM.

4-   Integrate ui design coding.

5-   Slice design PS to be HTML

6-   Create responsive design

7-   Publish the Graphical Design for Obtaining Commitment

8-   Perform Technical Review on the GD package by system analyst for functionalities
