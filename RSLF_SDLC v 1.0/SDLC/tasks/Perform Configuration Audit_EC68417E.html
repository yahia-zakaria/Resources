Task: Perform Configuration Audit
This task is designed to perform configuration audits for assisting the project team and senior management confirming that the resulting baselines from configuration management conform to the specified standards and plans.
Disciplines: Configuration Management
Purpose

The purpose of performing configuration audit is:

-          To determine that a baseline contains all required work products

-          To determine that a baseline meets requirements
