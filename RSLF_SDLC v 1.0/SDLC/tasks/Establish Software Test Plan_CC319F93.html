Task: Establish Software Test Plan
This task describes how to define the test plan and the specific techniques that will be employed, and outlines the test approach.
Disciplines: Testing (QC)
Purpose

The purpose of this task is to identify each specific technique that will be employed to enable the desired testing and outlining the workings of each technique including the types of testing it supports.
