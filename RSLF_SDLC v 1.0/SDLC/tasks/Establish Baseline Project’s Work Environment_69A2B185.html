Task: Establish Baseline Project’s Work Environment
This task prepares the final development and application lifecycle management environments for the project, where the development environment includes both process and tools.
Disciplines: Configuration Management

      Main Description

      The purpose of this task is to ensure that the final project environment is ready for the upcoming project phases and iterations. This includes process and tools.

       
      Steps
      Step 1. Customize the Project on the Server
      Step 2. Install, Customize and Prepare Clients that will connect to the Project
      Step 3. Set Up Configuration Management Environment
      Step 4. Review and Confirm Users Permissions
      Step 5. Create Integration Workspaces
      Step 6. Create Workspace in each Client
