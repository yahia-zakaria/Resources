Task: Initiate Project
This task describes how to staff and assign the team that will define the high-level lifecycle plan and the criteria for measuring project success in terms of high-level goals and objectives.
Disciplines: Project Management
Relationships
RolesPrimary Performer:
  • Projects Management Officer (PMO)
Additional Performers:
InputsMandatory:
  • Project Charter
  • Project SOW
    		Optional:
    • None
    Outputs
    • Project Statement Of Work
      Main Description
      Steps
      Step 1. Receive and review the new customer request Or Change request
      Step 2. Meet with PMO Manager to review the request and available resources, schedule, scope and technolog
      Step 3. If the request is approved Inform the Customer, PMO complete charter and assign PM
      Step 4. If the request is rejected inform the requester with clear justifications and reasons
