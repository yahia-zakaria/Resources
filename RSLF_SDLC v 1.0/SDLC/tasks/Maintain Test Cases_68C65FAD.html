Task: Maintain Test Cases
This task describes how to define the overall structure for the test suite implementation.
Disciplines: Testing (QC)
Purpose

The purpose of this task is to establish the structure in which the test suite and test cases implementation will reside as well as assigning responsibilities for test suite implementation areas and their contents by outlining the required test suites and test cases.
