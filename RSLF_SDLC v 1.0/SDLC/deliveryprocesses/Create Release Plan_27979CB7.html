Task Descriptor: Create Release Plan
Create a plan that includes the releases that can deliver to the client with specific start/end date.
Based on Method Task: Create Release Plan

Main Description
Steps
Step 1. Review the priorities of backlog
Step 2. Determine the sprint velocity
Step 3. Define the backlog into releases :
• Plans for each release.
• Subsequent sprints for the release(s).
• Plans for each sprint.
• Feature development within sprint.
• Individual tasks necessary to deliver a feature.
Step 4. Consider risk time and schedule vacation for the team in the release plan.
Step 5. Finalize and share product release calendar.
