Task Descriptor: Assess and Monitor Project Status and Progress
This task describes how to conduct status assessment in the regularly scheduled Report Status task.

Main Description

The Status Assessment identifies any exceptions that occur. Exceptions can be thought of as issues that are barriers to project progress (e.g. availability of equipment, availability of key individuals to answer questions, difficulty getting a decision made).

 

Steps
Step 1. Capture Work Status and Progress
Step 2. Collect Process Audits Status and Progress
Step 3. Collect Configuration Audits Status and Progress
Step 4. Prepare Project Status Report
Step 5. Communicate Project Status (Meeting or Offline)
Step 6. Record Decisions and Prepare Review Record (Meeting Minutes)
Step 7. Share Project Status Reports with Process Engineer each month (by email)
