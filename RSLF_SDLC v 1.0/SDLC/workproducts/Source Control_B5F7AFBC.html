Description
Main Description

The Source Control stores all the files and directories that are managed by the project's Configuration Manager. The Source Control is a global resource that will need to be accessed by most project team "clients" and “Workspaces”.

 

The number of files in any given Source Control will depend on the size of the machine on which the repository server is running, and the number of users expected to concurrently access data.
