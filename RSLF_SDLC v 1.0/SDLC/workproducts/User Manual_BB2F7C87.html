This artifact consists of materials that assist the end-user in learning, using, operating and maintaining the product.

Purpose

The purpose of this work product is to guide and support the user in how to use the product.

Description
Main Description

The user documentation gives instructions for using the software. Provide documentation for all types of users. Usually use cases are used as a basis for the user's guide. The user manual can be written by technical writers, with input from developers, or it can be written by the test team, whose members are likely to understand the user's perspective.
