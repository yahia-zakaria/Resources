Artifact: Project Plan
The purpose of the Project Plan is to gather all of the information necessary to control the project. It describes the approach to the development of the software, and is the top-level plan generated and used by the managers to direct the development effort.

Description
Main Description

The following people use the Development Plan:

The Project Owner and Project Manager

To plan the project schedule and resource needs and to track progress against the schedule.

 

Project Team Members

To understand what they need to do, when they need to do it, and what other activities they are dependent upon.
