Brief Outline

The following views should be available as minimim for viewing the current status of the work product:

<Task Name>

The name of the task.

<WBS Level Type>

The level type of the WBS item. It is one of the following:

- Phase

- Iteration

- Work Package

- Work Component

- Checkpoint

- Milestone

- Task

<% of complete>

The current status of a task expressed as the percentage of the taskâ€™s duration that has been completed.

<Duration>

The actual total span of working time for a task.

<Start>

The actual beginning date of a task.

<Finish>

The actual completion date of a task.

<Baseline Start>

The planned beginning date of a task.

<Baseline Finish>

The planned completion date of a task.

<Baseline Duration>

The planned total span of working time for a task.

<Duration Variance>

The difference between the baseline duration of a task and the actual duration.

<Resource Name>

Resource to be assigned to a task.
